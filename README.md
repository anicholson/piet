# piet_interpreter

an interpreter for the Piet esoteric programming language, written in Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  piet_interpreter:
    github: anicholson/piet_interpreter
```

## Usage
